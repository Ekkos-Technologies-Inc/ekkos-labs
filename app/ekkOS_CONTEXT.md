# apps/labs/app

The `apps/labs/app` directory serves as the experimental frontend and API interface for the ekkOS platform. It hosts interactive research demos, engineering visualizations, and specialized tools (such as the Ekk0 game and scorecard systems) that showcase the platform's capabilities in AI, reinforcement learning, and 3D engineering interfaces.

## Architecture
- **Runtime**: Node.js (Next.js 15+ App Router)
- **Entry points**: 
    - `(main)/page.tsx`: Root landing page for the labs environment.
    - `engineering/page.tsx`: Entry point for the engineering-focused visualization suite.
- **Key dependencies**: 
    - `next`: Framework for routing and server-side rendering.
    - `react`: UI library.
    - `framer-motion`: Used for complex animations in engineering components.
    - `three` / `@react-three/fiber`: Used for 3D engineering visualizations.
    - `packages-ekkos-sdk`: Core platform SDK for system integration.
    - `packages-gemini`: AI integration for chat-based interfaces.
- **Build system**: Next.js build pipeline (integrated into the monorepo CI).
- **Deploy target**: Vercel (standard for Next.js monorepo deployments).

## Key Files
- `(main)/ekk0/Ekk0Game.tsx`: Core component for the Ekk0 interactive game/demo.
- `engineering/LayerStack3D.tsx`: 3D visualization component for system layer stacks.
- `engineering/LiquidGlass.tsx`: UI component for high-fidelity glass-morphism effects.
- `engineering/ParticleNetwork.tsx`: Visual representation of network nodes/data flow.
- `engineering/WireframeGlobe.tsx`: 3D globe visualization for engineering demos.
- `api/ekk0/chat/route.ts`: Backend endpoint for Ekk0-related conversational AI.
- `api/scorecard/ingest/route.ts`: Endpoint for processing and ingesting scorecard data.
- `(main)/research/mit-rlm/page.tsx`: Dedicated page for MIT Reinforcement Learning research integration.

## API / Exports
The system exposes several internal API routes under the `/api` namespace:
- `POST /api/ekk0/chat`: Handles chat interactions for the Ekk0 module.
- `GET /api/ekk0/state`: Retrieves the current state of the Ekk0 game/session.
- `POST /api/engineering/chat`: Handles engineering-specific conversational queries.
- `POST /api/scorecard/ingest`: Ingests performance or research metrics.
- `GET /api/scorecard/latest`: Fetches the most recent scorecard data.
